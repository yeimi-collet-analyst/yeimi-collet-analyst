## 🚀 Proyectos Destacados

### 📦 Pipeline de Datos: Limpieza Automatizada de Inventario(https://github.com/yeimi-collet-analyst/inventory-data-cleaner)
* **El Desafío:** Una tienda de ropa recibía su catálogo en archivos de texto desordenados, con registros incompletos y errores tipográficos que colapsaban su sistema de inventario.
* **La Solución:** Creé un script defensivo en Python nativo que filtra registros vacíos en tiempo de ejecución, valida campos numéricos y automatiza la carga de datos limpios en una base de datos SQLite.
* **Tecnologías:** Python, SQLite, JSON.
